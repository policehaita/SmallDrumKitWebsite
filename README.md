Everything I learned resently are covered in this mini websites.

3. Playing sound on a websites
4. Using switch statements
5. Deeper knowledge about objects, their methods and dot notations
6. Using Keyboard Event Listeners to Check for Key Presses
7. Callbacks and How to Respond to Events
8. Adding Animation to Websites
